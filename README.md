# 清韵茶坊

《网站设计与开发》课程综合实验作品，一个以中国茶文化为主题的静态网站。

## 项目介绍

清韵茶坊围绕茶叶种类、品茶知识、茶具鉴赏、茶史文化、冲泡指南、茶与养生、茶礼茶俗等内容进行展示，并提供站内搜索、用户注册、用户登录、会员中心、留言交流和后台管理演示功能。

## 页面结构

- `index.html`：网站首页
- `tea.html`：茶叶种类
- `knowledge.html`：品茶知识
- `tools.html`：茶具鉴赏
- `culture.html`：茶史文化
- `brewing.html`：冲泡指南
- `health.html`：茶与养生
- `ceremony.html`：茶礼茶俗
- `search.html`：站内搜索
- `login.html`：用户登录
- `register.html`：用户注册
- `member.html`：会员中心
- `admin.html`：后台演示
- `sitemap.html`：网站结构
- `about.html`：项目说明

## 技术栈

- HTML5
- CSS3
- JavaScript
- localStorage 本地存储
- Vercel 静态网站部署

## 本地运行

直接用浏览器打开 `index.html` 即可运行。

演示账号：

- 用户名：`demo`
- 密码：`123456`

## Vercel 部署

本项目是纯静态网站，部署到 Vercel 时不需要构建命令。

推荐配置：

- Framework Preset：Other
- Build Command：留空
- Output Directory：留空或 `.`
